# [Grasp-of-Avarice](https://samchodev.itch.io/goa)
**Team 6 풀소유 뵤적단**

Grasp of Avarice는 유명한 해적 검은 수염의 해적선에서 눈을 뜬 주인공이 보물을 얻고 탈출하는 과정을 다룬 2D 퍼즐 플랫포머 게임입니다. A, S, D, Q, E, Space로 이루어진 직관적인 조작에 따라 주인공과 함께 세계 최고의 보물을 찾아 떠나봅시다!

Grasp of Avarice is a 2D puzzle platformer where the protagonist wakes up on the pirate ship of the legendary Blackbeard and must claim the ultimate treasure to escape. With intuitive controls using A, S, D, Q, E, and Space, set sail on an adventure to uncover the greatest treasure in the world!



## Controls

[A / S / D] Move left / down / right

[Space] Jump

[Q] Drop items

[E] Pick up items
